# TechSolutions - Professional Services Website

## Project Overview

TechSolutions is a responsive business website developed using HTML5, CSS3, and JavaScript.

The website provides information about the company, services, and contact details.

---

## Features

- Responsive Design
- Modern UI
- Navigation Bar
- Hero Section
- About Page
- Services Page
- Contact Page
- Contact Form Validation
- Dark Mode
- Scroll To Top Button
- Hover Animations
- Local Storage

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
